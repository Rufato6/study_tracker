# Study Tracker (SQLite + CLI + SRS + Reports)

A local study tracking tool for exam preparation.
It stores subjects, topics, study sessions, scores, and generates spaced-repetition review schedules.

## Features
- SQLite database (portable file)
- CLI to add subjects/topics, log sessions, record quiz results
- Spaced repetition (SM-2-like) scheduling for topic reviews
- Reports: weekly minutes, average scores, due reviews
- Optional GUI (Tkinter) and plots (matplotlib)
